欢迎使用本程序！

本程序是一个绿色程序，无安装，无卸载，不会读写系统注册表或任何用户信息，直接解压可用
这个说明文档主要解释一些使用中的常见问题，如果你是老用户，可以忽略本文！

1、程序不能运行报错问题
windows版本程序运行，依赖.net框架和vc运行时环境，大多数玩家的电脑上，由于以前安装过其他游戏的缘故，这些插件也许已经安装过了，因此，直接下载就可以玩。但有些小白用户的电脑上从未安装过这些插件，则必须安装这些插件或运行时环境，程序才能正常运行，请从上面链接下载。
IOS或Android版本不能运行，则主要与手机操作系统版本有关，需要升级操作系统之后，再试！

2、歌曲下载报错问题
本程序运行，下载歌曲，需要对磁盘进行读写操作。某些操作系统比如win10的某些版本，不允许程序写入C盘（系统盘），所以，我们推荐用户将程序放在D盘根目录下运行。注意：某些系统需要右键点击主程序，在菜单中选择“以管理员身份运行”！运行的过程中，如果360安全卫士等管家类软件，提示程序要求写权限，请点击“允许”，否则会提示下载错误。

3、延迟设定问题
使用不同的外设来进行游戏，延迟值的设定有所不同，比如使用键盘和触屏进行游戏，延迟通常设置为40左右即可，如果使用外接的MIDI设备（比如手卷鼓或者电鼓），那么延迟值根据硬件性能和工作条件不同设置在 180~320 之间。为了方便用户，在程序第一次运行的时候，会出现一个默认设备选择的界面，根据选择，程序会自动设定一个值，使程序的运行效果能基本匹配到设备，之后您还是可以通过系统设置界面，来手动调整延迟值

4、外接设备（比如外接的电鼓）敲击没反应，或者感觉不正常
由于各个外接设备的生产厂家不一，所以，每个鼓盘的信号默认值不尽相同，现在市面上比较好的电鼓主要是罗兰，那么我们的程序默认信号值设置，使用罗兰鼓的值，但用户仍然可以通过“系统设置”中的“鼓面设置”来重新设置每个鼓面的响应值。使程序能够完全脱离鼠标或键盘的操作，直接使用外接电鼓来操作程序。
